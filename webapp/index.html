BAZAAR

The Bazaar network is an open-source trade ecosystem, allowing for seamless integration with blockchains, exchanges, markets, supply chains, and banks.

We aim to provide your goods and services smart-contract guaranteed trade endpoints.

To facilitate connection between networks, there are a suite of network configuration tools allowing for deployment of a custom organization, or integration with existing systems:
• API
• Web Shop
• Blockchain
• ERC20 token
• Smart contract
• Custom Network

Trust on the system is provided through mutual validation.
